# Linear Timecode

A pure javascript implementation of a LTC encoder/decoder.


## Features

- [x] LTC decoder
    - [x] Support for 24, 25 and 30 fps
    - [ ] Support for 29.97 fps
    - [x] Support for drop-frame and color-frame
    - [ ] Support for user bits
- [ ] LTC encoder



## Usage

### Installation


- Run tests

- Build library
